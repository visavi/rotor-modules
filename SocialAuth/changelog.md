# Changelog

## 1.0.3
- Таймауты HTTP-запросов к OAuth-провайдерам (connect 5s / total 10s) — висящий провайдер больше не держит воркер 30 секунд
- Сетевой сбой (`ConnectionException`) при входе показывает сообщение об ошибке вместо страницы 500

## 1.0.2
- `socials.created_at` переведён с `TIMESTAMP` на `DATETIME` — единый тип дат во всём проекте (без проблемы 2038 и UTC-сдвига сессии). Модель `Social` уже на авто-`created_at` (`const UPDATED_AT = null`), типизирована константа
- Требует ядро 14.1.0

## 1.0.1
- Проверка чёрного списка при регистрации через соцсети: email и домен блокируют вход
- Blacklist-логин больше не блокирует, а перегенерируется
- Генерация логина использует полные 20 символов (было 16)
