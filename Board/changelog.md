# Changelog

## 1.0.2
- Поля `created_at`/`updated_at`/`expires_at` объявлений переведены с int (unix-таймстамп) на `DATETIME` (миграция с исторически-корректной конверсией таймзон)
- Сравнения и арифметика срока (`expires_at`) переведены на Carbon/`now()`; `$timestamps` оставлен выключенным — условная логика `updated_at` при перевыставлении сохранена
- Требует ядро 14.0.3
