# Changelog

## 1.0.1
- Поле `notebooks.created_at` переведено с int (unix-таймстамп) на `DATETIME` (исторически-корректная конверсия таймзон)
- Добавлена колонка `updated_at`: раньше `created_at` сбрасывался на каждом сохранении (= время правки). Теперь `created_at` = создание, `updated_at` = последняя правка; «Последнее изменение» в шаблоне берёт `updated_at`
- Включены авто-таймстампы вместо `$timestamps = false` (created_at/updated_at заполняются авто), убран ручной `SITETIME`
- При обновлении старых записей `updated_at` = прежний `created_at` (истинная дата создания была потеряна — `created_at` хранил время правки)
- Требует ядро 14.0.3
