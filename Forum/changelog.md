# Changelog

## 1.0.3
- Исправлены переводы (en, ua)
- Аудиоплеер в ленте топиков растянут на всю ширину (`w-100` вместо `img-fluid`) — пропадал в некоторых браузерах из-за css

## 1.0.2
- Убрано голосование из списков тем (новые, активные) — у темы нет рейтинга, брать его из последнего сообщения некорректно
- Исправлен fatal `formatNum(): null given` на пустом `lastPost` (`withDefault`)
- Рейтинг сообщений переведён на связь `poll` (`PollableTrait`) вместо ручного `leftJoin('polls')` — единый стиль с ядром, без ambiguous-колонок
- Исправлен вывод описания раздела на главной форума: несуществующая `plaintText` заменена на `renderText`
- Исправлен жирный текст описания раздела в админке (`/admin/forums`) — вынесен из `section-title`

## 1.0.1
- Рейтинг переведён на общий виджет голосования ядра (`app/_rating`)
- Добавлено голосование в списках тем (новые, активные) и сообщений (новые, мои)
- Единый стиль кнопок рейтинга и действий (`js-rating`, `js-actions`)
- Обновлён дизайн заголовков и расположения рейтинга в списках; мета (автор, дата, IP) вынесена из текста сообщения
- Требуется ядро 14.0.1
