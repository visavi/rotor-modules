# Changelog

## 1.0.2
- Поля `created_at`/`updated_at` переведены с int (unix-таймстамп) на `DATETIME` (миграция с исторически-корректной конверсией таймзон)
- Включены стандартные авто-таймстампы Eloquent (`created_at`/`updated_at` проставляются автоматически); `updated_at` в выводе показывается только при правке (`edit_user_id`)
- Требует ядро 14.0.3

## 1.0.1
- Кнопки действий (ответить, цитировать, жалоба, правка) переведены на единый круглый стиль ядра (`js-actions`)
