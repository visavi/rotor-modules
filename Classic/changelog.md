# Changelog

## 1.0.1
- Календарь: навигация по месяцам (стрелки ‹ ›, параметр `?calendar=Y-m`); `getCalendar()` принимает период `Y-m` вместо timestamp
- Подсветка текущего дня только при открытом текущем месяце
- Заливка цветом всей ячейки: текущий день и дни с новостями (раньше красился только номер)
- Сетка с общими бордерами (вид таблицы), скруглённые внешние углы
- Название месяца в именительном падеже из локали Carbon, без отдельных переводов
- Стили календаря вынесены из ядра в модуль (`resources/assets/css/calendar.css`), подключаются через `head`-хук
