# Changelog

## 1.0.1
- Поля дат сообщений (`chats`: `created_at`/`updated_at`) переведены с int (unix-таймстамп) на `DATETIME` (исторически-корректная конверсия таймзон, пересоздание индекса `created_at`)
- Сравнения/арифметика дат переведены на Carbon/`now()`; включены авто-таймстампы (`$timestamps`), ручные `created_at`/`updated_at` убраны — маркер «изменено» висит на `edit_user_id`, не на `updated_at`
- Маркер новых сообщений (`statsNewChat()` / `users.newchat`) переведён с таймстампа на `id` последнего сообщения — индикатор больше не зависит от формата дат (одноразово: при первом заходе в чат после обновления маркер пересчитается)
- Требует ядро 14.0.3
