# Настройки сайта

## Доступ к настройкам

AdminPanel → Настройки (раздел в левом меню).

Настройки хранятся в базе данных. В коде доступны через хелпер:

```php
setting('key')              // получить значение
setting('key', 'default')   // со значением по умолчанию
```

В Blade-шаблонах:

```blade
{{ setting('title') }}
{{ setting('description') }}
```

## Основные настройки

### Общие
| Ключ | Описание |
|------|----------|
| `title` | Название сайта (отображается в `<title>`) |
| `description` | Описание сайта (мета-тег) |
| `email` | Email администратора |
| `url` | URL сайта |
| `language` | Язык по умолчанию (`ru`, `en`) |
| `theme` | Активная тема оформления |
| `timezone` | Часовой пояс |

### Регистрация и пользователи
| Ключ | Описание |
|------|----------|
| `registration` | Включить регистрацию |
| `confirm` | Подтверждение email при регистрации |
| `invite` | Регистрация только по приглашениям |
| `max_avatar_size` | Максимальный размер аватара (KB) |

### Загрузка файлов
| Ключ | Описание |
|------|----------|
| `max_image_size` | Максимальный размер изображения (KB) |
| `max_file_size` | Максимальный размер файла (KB) |
| `image_quality` | Качество сжатия JPEG (1–100) |

### Безопасность
| Ключ | Описание |
|------|----------|
| `captcha` | Тип капчи (`math`, `gd`, `recaptcha`, `none`) |
| `spam_filter` | Включить антиспам |
| `antimat` | Включить фильтр нецензурных слов |
| `flood_time` | Время между сообщениями (секунды) |

### Производительность
| Ключ | Описание |
|------|----------|
| `cache_time` | Время кеша (секунды) |
| `paginate` | Количество записей на странице |

## Сброс кеша настроек

После изменения настроек через AdminPanel кеш сбрасывается автоматически. Для ручного сброса:

```bash
php artisan cache:clear
```

Или из кода:

```php
clearCache();
clearCache('settings');  // только настройки
```
